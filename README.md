# Pipeline de Datos - Ingesta

## Descripción
Este proyecto implementa un proceso de ingesta de datos automatizado, que toma un archivo CSV desde una fuente local y lo almacena en una estructura organizada dentro de un pipeline de datos.

## Funcionalidades
- Lectura de archivo CSV
- Copia automatizada a carpeta /data/raw
- Generación de logs del proceso
- Conteo de registros procesados

## Ejecución

```bash
python src/ingesta.py
